# 🎬 Movie Ticket Booking System

## 🚀 Features
- User login & booking system
- Seat selection with real-time UI
- Admin dashboard with analytics
- Seat locking & unlocking system
- Payment UI (UPI, Card, NetBanking)
- Ticket confirmation page

## 🛠 Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL
